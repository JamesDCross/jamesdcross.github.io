Instructions

This App will only run in Windows 10. UWP's are designed to run only on the Windows App Store, a sideload is used for testing. Right click on the Add-AppDevPackage file and run with the Powershell to install.

How the App works

The app parses the movie file titles in the folder it is directed to, using regex and other techniques. The results of the parsing are used to make calls out to the MovieDB Web API https://developers.themoviedb.org The JSON received is then deserialized into instances of classes for use in C#. Databinding is then used to establish the connection between the call results and the UWP UI.
